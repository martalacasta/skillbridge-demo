## What the Project Does

SkillBridge is a lightweight, frontend-only web application that connects companies and individuals with vetted technical experts and students for short-term technical projects. The landing page showcases the platform's services, how it works, and example projects, providing a modern, responsive, and easy-to-navigate experience with interactive features for user engagement.

## Why the Project Is Useful

- **Showcases project opportunities**: Highlights real-world example projects and the value of short-term technical engagements.
- **Modern, responsive design**: Works seamlessly on desktop and mobile devices.
- **Interactive user experience**: Includes light/dark mode toggle, smooth anchor navigation, and a project submission form.
- **No backend required**: Easy to deploy as a static site for demos or informational purposes.
- **Clear user journey**: Explains how SkillBridge matches users with technical talent.

## Key Features

- **Hero section** with project mission and call to action
- **Services overview** (Expert Matching, Short-term Projects, Flexible Engagement)
- **Step-by-step "How It Works" guide** with visual flow
- **Example projects** with tags, status, and detailed descriptions
- **Get Started form** for project submission with validation
- **Light/Dark mode toggle** for personalized viewing experience
- **Responsive navigation** with anchor links to sections

## How Users Can Get Started

### Prerequisites
- [Node.js](https://nodejs.org/) (optional, only if you want to use a static server)
- Or any static file server (e.g., Python's `http.server`)

### Installation & Local Development

1. **Clone the repository:**
	```sh
	git clone https://github.com/martalacasta/skillbridge-demo.git
	cd skillbridge-demo
	```
2. **Start a local server:**
	- With Python 3:
	  ```sh
	  python -m http.server 8000
	  ```
	- Or with Node.js (if installed):
	  ```sh
	  npx serve .
	  ```
3. **Open your browser:**
	Visit [http://localhost:8000](http://localhost:8000) to view the landing page.

### Usage Example

- **Browse the landing page** to learn about SkillBridge's services and see example projects.
- **Toggle light/dark mode** using the button in the header to switch your viewing preference.
- **Use anchor navigation** to quickly jump to different sections (Services, How It Works, Projects, Get Started).
- **Submit a project** via the "Get Started" form to propose a new technical project.
- **Contact information** is available in the footer for partnership inquiries.
- The site is static and does not require any backend or database setup.
